# 🎲 Яцзи (Yatzy) — Telegram Mini App

Multiplayer Yatzy для двух игроков, интегрированный как Telegram Mini App, с локальным режимом против AI-бота, продвинутой отладкой и интерактивными сюрпризами.

## Стек

| Слой | Технология |
|------|-----------|
| Frontend | React + TypeScript + Vite |
| Backend | Node.js + Express + Socket.io |
| Протокол | WebSocket (real-time) |
| Интеграция | Telegram Web App SDK |
| Реклама | Google IMA / Yandex YAN (динамический выбор) |

## Основные особенности

- **Мультиплеер в реальном времени**: Быстрый запуск игр по коду комнаты через Socket.io.
- **Оффлайн-режим против бота**: Умный ИИ-противник с оптимизированной тактикой выбора комбинаций.
- **Восстановление сессии (Session Recovery)**: Защита от разрывов связи и сворачивания приложения.
- **Интеграция с Telegram**: Получение имени и аватара пользователя, нативная вибрация (haptics) при ключевых действиях.
- **Визуальные эффекты**: Анимированный 3D-эффект вращения кубиков, золотая подсветка комбинации Яцзи и праздничный оверлей с конфетти (с плавным растворением).
- **Система эмодзи-реакций**: Удобная отправка 11 видов реакций с отображением в центре экрана и бейджами аватаров.
- **Монетизация**: Бонусные броски за просмотр видеорекламы с автовыбором провайдера под ГЕО.

## Быстрый старт

### Установка зависимостей

```bash
# Сервер
cd server && npm install

# Клиент  
cd client && npm install
```

### Запуск в разработке

**Сервер** (порт 8000 по умолчанию, с возможностью настройки):
```bash
cd server
npm run dev
```

**Клиент** (порт 5173):
```bash
cd client
npm run dev
```

Открой http://localhost:5173 в браузере.

## Деплой для Telegram Mini App

### Сервер
Задеплой на любой VPS / Railway / Render / Koyeb (в проекте настроена конфигурация и Dockerfile).

### Клиент
1. Обнови `VITE_SERVER_URL` в `client/.env`:
   ```
   VITE_SERVER_URL=https://yatzy.example.com
   ```
2. Собери:
   ```bash
   cd client && npm run build
   ```
3. Размести папку `dist/` на Vercel / Netlify / Nginx.

### Telegram Bot
1. Создай бота через [@BotFather](https://t.me/BotFather)
2. Введи `/newapp` → укажи URL твоего задеплоенного клиента.
3. Поделись ссылкой на Mini App.

## Игровые правила

**Ходы:** каждый игрок бросает кубики до 3 раз за ход, удерживая нужные.  
**Очки:** выбери категорию для записи очков после бросков.

### Подсчёт очков

| Секция | Категория | Очки |
|:-------|:----------|:-----|
| **Верхняя** | Единицы … Шестёрки | Сумма кубиков с выбранной гранью |
| | **Бонус** | **+35** (если сумма верхней секции ≥ 63) |
| **Нижняя** | Три одинаковых | Сумма всех кубиков |
| | Четыре одинаковых | Сумма всех кубиков |
| | Фулл-хаус | 25 |
| | Малая улица | 30 (4 последовательных значения) |
| | Большая улица | 40 (5 последовательных значений) |
| | Яцзи (Yatzy) | 50 (5 одинаковых) |
| | Шанс | Сумма всех кубиков |



### 🔥 Комбо-бонусы (Combo Achievements)

В игре реализована система фиксированных дополнительных бонусов за сбор определенных наборов комбинаций в таблице очков:
- 🏃‍♂️ **Уличный забег (+20 очков):** Требуется закрыть категории «Малая улица» и «Большая улица» со счетом больше 0.
- 👑 **Сила равных (+25 очков):** Требуется закрыть категории «Три одинаковых», «Четыре одинаковых» и «Фулл-хаус» со счетом больше 0.
- 🧸 **Детский сад (+12 очков):** Требуется полностью заполнить категории «Единицы», «Двойки» и «Тройки» в верхней секции так, чтобы их суммарный счет составил 10 или более очков.

Прогресс выполнения каждого комбо наглядно отображается в нижней части таблицы результатов (например, `1/2`, `2/3`, а в случае провала условий «Детского сада» выводится значок `✕`).

### 💬 Эмодзи-реакции (Reactions)

В игре реализована система быстрых реакций для взаимодействия между игроками:
- **Интерфейс**: Кнопка `💬` находится в правом нижнем углу экрана. Открывает панель из 11 эмодзи: `👍`, `🔥`, `🎉`, `💩`, `😎`, `😢`, `😘`, `🖤`, `💅`, `🥂`, `😈`.
- **Отображение**: Выбранный эмодзи отображается в укрупненном виде по центру экрана с эффектом зума и легким поворотом. В углу реакции крепится маленький бейдж с аватаром отправившего игрока. При отправке нескольких реакций подряд они немного рассеиваются по осям X/Y, чтобы не перекрывать друг друга.
- **Ограничения**: Лимит отправки — 1 реакция раз в 2.5 секунды.
- **Игра против бота**: Бот реагирует на ваши эмодзи с шансом 70% (задержка 0.8–1.5с). Также бот шлет автоматические эмодзи при ярких событиях: `🔥`/`😎` при выпадении Yatzy и `😢` при вычеркивании категории на 0 очков.

## Специальные режимы и секретные коды

В поле **"Код комнаты"** при подключении можно ввести специальные коды для перехода в скрытые режимы:

### 🛠️ Дебаг-комната разработчика (Код: `DEBUG`)
Запускает локальный тестовый матч против бота с панелью инструментов разработчика для отладки игровой логики.
- **Выставление кубиков:** Ручное назначение любого значения каждому из 5 кубиков для отладки сложных комбинаций.
- **Откат хода (Undo):** Пошаговое возвращение игры на любое количество ходов назад.
- **Быстрый бонус (+63):** Кнопка мгновенной записи 63 очков в верхнюю панель для тестирования бонуса +35.
- **Случайное заполнение:** Автозаполнение таблицы случайными значениями.
- **Мгновенное завершение:** Кнопки принудительного выигрыша или проигрыша.

### 🎂 Интерактивный сюрприз (Код: `SONYA` — Временно отключен)
*Секретный режим деактивирован в Lobby.tsx, но исходный код SurpriseRoom сохранен в репозитории.*
Особый праздничный экран-поздравление с интерактивными элементами:
- Красочный векторный торт (SVG) с четырьмя свечами.
- Механика "задувания": свечи тухнут по тапу с вибрацией (Telegram Haptics).
- При задувании всех свечей срабатывает взрыв конфетти и открывается скрытое поздравительное сообщение.
- Через 15 секунд появляется кнопка возврата в лобби.

## Технические особенности

### 🔄 Мягкое отключение и автовосстановление сессии (Session Recovery)
- При кратковременном отключении игрока (сворачивание Telegram, входящий звонок, сбой сети) сервер сохраняет его сессию в течение **3 минут**.
- Противник получает уведомление с таймером ожидания.
- При повторном открытии Mini App игрок мгновенно и бесшовно возвращается в игру с сохранением всех очков и состояния бросков.
- Если игрок не вернулся за 3 минуты, его противнику засчитывается автоматическая победа.

### 🎥 Интеграция рекламы (Ad Services)
- В режиме против бота игроку доступен **1 дополнительный бонусный бросок** за просмотр короткого видеоролика.
- Интеграция умеет динамически переключаться между **Google Interactive Media Ads (IMA)** и **Yandex Advertising Network (YAN)**, выбирая оптимального провайдера на основе локали и ГЕО игрока.

## Socket API

| Событие (client → server) | Данные | Описание |
|----------------------------|--------|----------|
| `create_room` | `{ name, avatar }` | Создать комнату |
| `join_room` | `{ code, name, avatar }` | Войти в комнату |
| `roll_dice` | — | Бросить кубики |
| `toggle_hold` | `{ index }` | Зафиксировать/снять кубик |
| `score_category` | `{ category }` | Записать очки |
| `rematch` | — | Реванш |
| `surrender` | — | Сдаться (завершить игру досрочно) |
| `leave_room` | — | Выйти из комнаты |
| `reconnect_session` | `{ code, sessionId }` | Попытка переподключения к сессии после обрыва связи |
| `send_reaction` | `{ emoji }` | Отправить эмодзи-реакцию |

| Событие (server → client) | Описание |
|---------------------------|----------|
| `room_created` | Комната создана, код |
| `game_started` | Второй игрок подключился |
| `game_state` | Обновление состояния игры |
| `game_over` | Игра окончена, победитель |
| `player_disconnected` | Противник окончательно покинул игру |
| `player_temporarily_disconnected` | Противник временно отключился (запущен таймер ожидания 3м) |
| `player_reconnected` | Противник успешно переподключился |
| `reconnect_failed` | Ошибка восстановления сессии |
| `reconnected_successfully` | Успешное восстановление собственной сессии |
| `receive_reaction` | Переслать реакцию от оппонента `{ senderId, emoji }` |
| `error` | Ошибка |

# План реализации визуального эффекта Яцзи

## Идеи эффектов
1.  **Конфетти:** Полноэкранный взрыв конфетти при обнаружении комбинации из 5 одинаковых кубиков.
2.  **Золотое свечение кубиков:** Все 5 кубиков начинают пульсировать золотистым светом.
3.  **Экранная надпись "YATZY!":** Крупный анимированный текст, появляющийся по центру экрана.
4.  **Звуковой эффект:** (Опционально) Короткий торжественный звук.

## Технический стек
*   **CSS Animations:** Для свечения кубиков и появления текста.
*   **Canvas Confetti:** Легкая библиотека для взрыва частиц (или простая CSS-реализация, если хотим без зависимостей).
*   **React State:** Для запуска эффекта в момент изменения состояния кубиков.

## Список задач
1.  **Инсталляция (опционально):** `npm install canvas-confetti` в папке `client`.
2.  **Компонент эффектов (`client/src/components/YatzyOverlay.tsx`):**
    *   Создать новый компонент для отображения надписи и управления конфетти.
3.  **Стилизация (`client/src/components/YatzyOverlay.css`):**
    *   Добавить ключевые кадры (keyframes) для анимации текста и свечения.
4.  **Интеграция в `GameBoard.tsx`:**
    *   Добавить проверку: если `dice` содержит 5 одинаковых значений — показывать оверлей.
5.  **Обновление `Die.tsx`:**
    *   Добавить пропс `isYatzy`, который активирует CSS-класс со свечением.

## Ожидаемый результат
При выбросе 5 одинаковых кубиков экран на мгновение «взрывается» праздничными искрами, а кубики подсвечиваются, подчеркивая важность момента.

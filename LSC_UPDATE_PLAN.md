# План обновления механики Lucky Streak Multiplier (LSC)

## Цель
Изменить условия накопления и сброса множителя LSC для соответствия новым правилам:
1.  **Исключение Chance:** Категория "Шанс" (Chance) больше не участвует в накоплении комбо и на неё не действует текущий множитель.
2.  **Сброс при нуле:** Если любая комбинация Нижней секции закрывается на 0 (вычеркивается), стрик сбрасывается до x1.0.
3.  **Сброс при выборе Верхней секции:** (Существующее) Выбор любой категории из верхней секции сбрасывает стрик.

## Список задач
1.  **Backend (`server/src/index.ts`):**
    *   Обновить обработчик `score_category`.
    *   Добавить условие: `category !== 'chance'`.
    *   Добавить условие: `score > 0` для продолжения стрика в Нижней секции.
    *   Обеспечить сброс `lscStreak` и `lscMultiplier`, если условия не выполнены.

2.  **Frontend (`client/src/hooks/useLocalGame.ts`):**
    *   Синхронизировать логику локальной игры с серверной.
    *   Обновить функцию `selectCategory`.

3.  **Документация (`README.md`):**
    *   Обновить описание правил LSC.

4.  **Тестирование:**
    *   Проверить, что Chance не умножается и сбрасывает стрик.
    *   Проверить, что "вычеркивание" (0 очков) в Нижней секции сбрасывает стрик.
    *   Проверить, что успешные комбинации (кроме Chance) продолжают растить стрик.
